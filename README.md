Project Title: Crop Recommendation System
Technologies Used: Python, scikit-learn, Pandas, Flask, HTML, CSS.
How it Works : Takes user input for season and soil mineral levels, processes it using a trained model, and outputs recommended crops.
Installation/Setup Instructions: git clone, pip install -r requirements.txt, python app.py.
Dataset (if applicable): Our dataset for training the crop recommendation model is a hybrid collection. It includes publicly available agricultural data (e.g., historical crop yields, soil characteristics, seasonal patterns) sourced from reputable online platforms, significantly augmented by synthetic data generated using AI tools to cover a wider range of scenarios and fill potential data gaps.
Future Enhancements: Integrate Weather Data, Location-Specific Recommendations, Advanced Soil Analysis, Water Availability/Management, Crop Rotation Recommendation, Nutrient Deficiency Diagnosis.
